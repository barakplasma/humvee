This folder is an Obsidian-compatible reference vault for the Humvee A2 Driver
Trainer. Use these notes before returning to raw PDFs or ad hoc web searches.

## Rules For Future Edits

- Keep the game framed as a simplified training aid, not official instruction.
- Add exact source identity, date, page/figure/paragraph when known.
- Mark inferred/simplified gameplay thresholds clearly.
- Prefer updating these distilled notes before changing game behavior.
- If a raw source contradicts a note here, update the note and cite the source.
